# method scope 
 - means variable can be used after the line of declaration , and can be used  only in that particular method where it is declared.

# Block scope  
- means variable can be used in that particular block where it is declared.
eg:-     {
           int s = 6 ;
           
               }
          System.out.println (s)

          here s is not determined outside the scope (braces)

# class scope
-  access modifiers are used.
types of access modifier :- #private , #public ,#protected
