// operators in java 

  operator is a symbols that tell compiler to perform some operation 
     
    sum = a + b    ( this is an expression)

     #  a,b are operands  
     #   + is operator



_________________ types of operator _____________________________________

1. Arithmatic Operators(binary/unary)/ternary
2. Relational operators
3. Logical operator
4. Bitwise operator ( Bit manupulation)
5. Assignment operator


   
   
   
// 1.Arithmatic operators

          // Binary ( minimum 2 operands) =>   + , - , * , / , %

          // Unary oprator (unit operand) => ++ , --
     
             ++a =>  pre increment 
             --a =>  pre decrement
             a++ =>  post increment
             a-- =>  post decrement 
    

// 2. Relational operators
         
         ==   equal to
         !=   inequality (not equalto)
         >    greater than
         <    less than
         <=   lessthan equalto
         >=    greaterthan equalto


// 3. Logical operators
          
          &&  => logical and 
          ||  => logical or
          !   => logical not

// 4. Assignment operator
       
      //   +=  =>   a += 2
      //   -=  =>   a -= 3
      //   *=  =>   a *= 2
      //   /=  =>   a /= 5
